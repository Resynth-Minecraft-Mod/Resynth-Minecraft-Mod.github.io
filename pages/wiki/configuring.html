Configuring Resynth
A big goal of Resynth is to be as configurable as possible. This means that most "values" used in the mod (such as drop chances, growth times, drop rates as well other various settings) can be changed though either the mods In-Game configuration menu (Accessed by going to the pause menu -> Mods -> Resynth) or by editing the mods configuration file ( Resynth.cfg). Most values can be changed though the configuration menu without requiring a world/Minecraft restart so you can change values without having to quit the game.

Basic Settings
This section covers the most basic and important configuration settings.

General Plant Settings
Settings that affect every plant added by Resynth.

Config Name Description Value Type Range Submenu Title Default Value
Enable Bonemeal Global toggle to disable growing all Resynth plants with bonemeal. Boolean N/A General Plant Settings false
Enable Seed Drops From Ore Global toggle to disable any ores dropping seeds when blown up. Boolean N/A General Plant Settings true
Enable Seed Drops From Organic Ore Global toggle to disable any Organic Ores dropping seeds when blown up. Boolean N/A General Plant Settings true
Enable Seed Drops From Mobs Global toggle to disable mobs dropping seeds when killed. Boolean N/A General Plant Settings true
Enable Seed Drops From Produce Global toggle to disable any produce items (Shards & Bulbs) turning into seeds. Boolean N/A General Plant Settings true
Enable Growth Global toggle to disable Resynth plants from growing. Effectively disables the mod. Boolean N/A General Plant Settings true
Enable Smokey Plants Global toggle to disable plants spawning smoke particles around them. Boolean N/A General Plant Settings true
