Minecraft 1.14.4

Version 3.0.0

A direct port to 1.14.4 from 1.13.2 with some additions.

warning Integration with other mods as been added back - This includes Hwyla integration and plant sets for a few new mods. The One Probe integration has yet to be added. Biochemical plants can be made for modded resources as well, this means Resynth can grow mob drops from other mods now.

Changes

error Changed Mineral Stone world generation again - The way Mineral Stone generates in the world has been changed slightly. It now generates more like vanilla ores and it's spawn frequency has been increased.

Additions

add_circle Effects added when seeds spawn - When seeds spawn, they'll now display particle effects and play a sound to let you know they've spawned (only works on singleplayer).
add_circle Added plants for additional mods - With support for modded plants back, a few new plants have been added for the following mods:

Minecraft 1.13.2

Version 2.0.0 (Beta)

The Big One

warning A Massive Rewrite - Due to the large changes to forge in the 1.13 update, a large portion of Resynth had to be rewritten. This gave the opportunity to rewrite other large portions of Resynth which could have been written better, which has resulted in many changes, some of which may be omitted. Hopefully these changes will result in a better experience.
warning Large changes to the Mineral Hoe - The Mineral Hoe has been rewritten from the ground up and as such as changed drastically. It now operates slightly differently and a few new features have been added to it.

Changes

error Changed how seeds are obtained from Bulbs - Obtaining seeds from Bulbs (Biochemical plant produce) has changed slightly. Instead of being thrown, they're now smashed (by holding right-click) for a chance at getting more seeds.
error Changed Crystalline Plant Textures - The plant textures for Crystalline plants was changed to a new design that doesn't mimic Vanilla Wheat.
error Changed World Generation - Mineral Stone and Mystical Seed Pods world generation has changed. Mineral Stone now spawns a bit higher up and more frequently, while Mystical Seed Pods generate in larger numbers.
error Changed Configuration - The whole configuration system for Resynth was rewritten. This has unfortunately resulted in the loss of the in-game configuration GUI. However, configuration settings are now more consistent and descriptive. Plant config settings now have their own config file.
error Changed Tooltips - Tooltips have been completely rewritten to be more descriptive, provide more information, be more consistent and contain less errors.
error Biochemical seeds can be dropped from multiple mobs - Seeds for Biochemical plants can now be obtained from multiple different mobs.

Additions

add_circle Animated Textures - Animated textures for the Mineral Hoe and Mineral Crystal were added.
add_circle Growth Feature to the Mineral Hoe - The Mineral Hoe can now grow any plant (Resynth or Vanilla) when used in creative mode.
add_circle Information Feature to the Mineral Hoe - The Mineral Hoe can now be used to check the growth stage of any Resynth plant as well as the Mineral Content of Mineral Soil.
add_circle Plant Yield - Crystalline and Biochemical plant types can now drop multiple produce items when harvested.
add_circle Two new plants - Added two new Biochemical plant sets: Experience Bottle (For Bottle o' Enchanting) and Nautilus Shell.

Removals

remove_circle Modded Plant Sets - With the lack of mods for Minecraft 1.13 as well as the release of Minecraft 1.14, plants for other mods have unfortunately been removed until they can be added back.
remove_circle Hwyla and The One Probe Support - Support for Hwyla and The One Probe has been removed with plans to add it back in the 1.14 update.
