label Added: Support for The One Probe.

label Added: Clay plant.
label Added: End Stone plant.
label Added: Sand plant.
label Updated: Mineral Soil texture (animated).
label Updated: Mineral Stone texture (animated).
label Fixed: Resynth crashing when run on a server (Thanks to Mozza7 for reporting this issue).
label Fixed: Various other bugs and issues that cause crashes on servers.
label Fixed: Dialog popup opening when blowing up Organic Ores.

Version 1.3.1

A lot of changes, fixes and improvements.

label Added: Mineral Hoe (to replace the old way of turning dirt into Mineral Soil).
 label Added: Yellorite Plant for Big Reactors/Extreme Reactors.
 label Added: Sound effects when seeds are dropped by a mob or an ore.
 label Added: Hwyla interaction.
 label Changed: The size of the Mineral Crystal texture.
 label Changed: All Straw items are now called shards.
 label Changed: The textures of Straw/Shard items.
 label Changed: Mineral Soil now drops Dirt and a Mineral Crystal when broken instead of the Mineral Soil block.
 label Changed: Mineral Rocks texture.
 label Changed: Mineral Rocks name to Mineral Rock.
 label Changed: Tooltips so they now provide more accurate information with less spelling/grammar errors.
 label Changed: All Organic Ore blocks now smelt into the original ore instead of the resource. For example, Organic Iron Ore now smelts into Iron Ore instead of Iron Ingots.
 label Fixed: Broken name for rabbit foot seeds.
 label Fixed: Broken name for rabbit foot bulb.
 label Fixed: Small bugs and spelling errors.
 label Fixed: The logs being flooded with block and item names.
 label Removed: The ability to turn dirt/grass into Mineral Soil with a Mineral Crystal.
 label Removed: The Mineral Soil crafting recipe.
 label Updated: Forge build version from 1.12.2-14.23.4.2705 to 1.12.2-14.23.5.2768.
 label Updated, fixed, refactored and cleaned: The code base.
