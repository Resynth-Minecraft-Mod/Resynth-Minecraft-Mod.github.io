[Download]

Resynth 6.0.0 (1.16.4)

Requires MinecraftForge 35.1.37 or newer

Date Published: February 1, 2021 -- MD5 Checksum: 3DF807D89A393D98FA24D6DE48AF68B8

Brings a lot of fixes and improvements to many of Resynth's core features, for a more stable and feature-rich experience. This includes the reintroduction of world-generation for ores and flowers, many new plants for some popular mods, and many various other improvements. Finally, most known issues and bugs have been fixed as well.
[Download]


Additions
add_circle_outline World-Generation is back - Underground ore generation is back along with natural flower generation, and it's better now.
add_circle_outline The One Probe integration - Resynth now supports The One Probe as well as Hwyla.
add_circle_outline Configurable produce output - The amount of resources and experience obtained when smelting plant produce can now be changed in the configuration files.
add_circle_outline New plants for popular mods - Lots of new plants have been added for some very popular mods, meaning the following resources can now be grown:

  • Ancient Debris (replaces the Netherite Scrap Plant)
  • Applied Energistics 2:
    • Certuz Quartz
    • Charged Certuz Quartz
  • Thermal Foundation:
    • Apatite
    • Cinnabar
    • Ruby
    • Sapphire
    • Sulfur
    • Copper
    • Lead
    • Nickel
    • Silver
    • Tin
  • Immersive Engineering:
    • Aluminium
    • Copper
    • Lead
    • Nickel
    • Silver
    • Uranium

Changes
error_outline Ambient-light based plant growth - Plant growth is now a lot more sensitive to ambient-lighting. Low light now negativly impacts plant growth heavily, however, very bright light can now also seriously speed up plant growth.
error_outline Plant-Seeds & Plant-Produce Tooltips - The tooltips for Plant items, such as Seeds and Produce, has been redesigned to be much better. The "PlantSet Stats" part of the tooltips now work on dedicated servers as well.
error_outline Plant & PlantSet Recipes - All recipes related to Plants and their items are now handled in code. This means recipes can be more easily changed. This should also seriously help fix the spamming of logs.
error_outline Dedicated Server Improvements - Dedicated servers and the clients that connect to them can now use PlantSet Item Tooltips, as well as see and hear audio/visual effects.
error_outline Better English localizations - A lot of the English based translations have been changed, improved, added, and just made better. This means better descriptive tooltips.
error_outline Better Logging - Console Logging has been cleaned up, so now the logs will have less spam, will include more important information, and is treated with more care in general.
error_outline Too many tiny changes - So much of Resynth's internal structure has changed since the last release, resulting in dozens of very minor changes. All the changes can be found in the list of code commits.

Removals
remove_circle_outline Old and broken PlantSets - All the recent changes have made a small amount of PlantSets no longer relevant, and they had to be removed. The following PlantSets are no longer apart of the mod:
  • Netherite Scrap (Replaced by Ancient Debris)
  • Nether-Gold (from the mod: Basic Nether Ores)

Fixes
check_circle_outline Fixed Datapack Errors ( Issue#36 ) - Fixed problems where various resources were not correctly registered to the game, which could cause weird and potentially dangerous behaviour.
check_circle_outline Fixed Console Errors ( Issue#30 ) - Fixed a lot of (many minor) errors that would show up in the console, as well as reduced error spamming by a large margin.
check_circle_outline Fixed crashes when loading/creating worlds - Fixed a couple rare errors and crashes that would show up when creating or loading a world.
check_circle_outline Lots of tiny bugs - Fixed a lot of other very minor bugs.

Resynth 2.0.0 Beta (1.13.2)

Rewrote a significant portion of Resynth due to the changes to Forge in the 1.13.2 update. This has resulted a lot of changes, some of which may be omitted.
Date Published: Sep 2, 2019
download

Additions
add_circle_outline Animated Textures - Animated textures for the Mineral Hoe and Mineral Crystal were added.
add_circle_outline Added a Growth Feature to the Mineral Hoe - The Mineral Hoe can now grow any plant (Resynth or Vanilla) when used in creative mode.
add_circle_outline Information Feature to the Mineral Hoe - The Mineral Hoe can now be used to check the growth stage of any Resynth plant as well as the Mineral Content of Mineral Soil.
add_circle_outline Plant Yield - Crystalline and Biochemical plant types can now drop multiple produce items when harvested.
add_circle_outline Two new plants - Added two new Biochemical plant sets: Experience Bottle (For Bottle o' Enchanting) and Nautilus Shell.

Changes
error_outline Large changes to the Mineral Hoe - The Mineral Hoe has been rewritten from the ground up and as such has changed drastically. It now operates slightly differently and a few new features have been added to it.
error_outline Changed how seeds are obtained from Bulbs - Obtaining seeds from Bulbs (Biochemical plant produce) has changed slightly. Instead of being thrown, they're now smashed (by holding right-click) for a chance at getting more seeds.
error_outline Changed Crystalline Plant Textures - The plant textures for Crystalline plants was changed to a new design that doesn't mimic Vanilla Wheat.
error_outline Changed World Generation - Mineral Stone and Mystical Seed Pods world generation has changed. Mineral Stone now spawns a bit higher up and more frequently, while Mystical Seed Pods generate in larger numbers.
error_outline Changed Configuration - The whole configuration system for Resynth was rewritten. This has unfortunately resulted in the loss of the in-game configuration GUI. However, configuration settings are now more consistent and descriptive. Plant config settings now have their own config file.
error_outline Changed Tooltips - Tooltips have been completely rewritten to be more descriptive, provide more information, be more consistent and contain less errors.
error_outline Biochemical seeds can be dropped from multiple mobs - Seeds for Biochemical plants can now be obtained from multiple different mobs.

Removals
remove_circle_outline Modded Plant Sets - With the lack of mods for Minecraft 1.13 as well as the release of Minecraft 1.14, plants for other mods have unfortunately been removed until they can be added back.
remove_circle_outline Hwyla and The One Probe Support - Support for Hwyla and The One Probe has been removed with plans to add it back in the 1.14 update(s).
